- `wezterm` config is intended to be used on a windows system.
- `fish` set as the default shell.

> [!Important]
>
> Make sure to update the paths configured which can be listed using the command below.
>
> ```fish
> rg "[\/\\\\]+home[\/\\\\]+jeswins" --hidden
> ```
